🎞 Letterboxd Sync NEW in 1.1

Every user on this server can now sync their Letterboxd ratings into StarTrack. Click the ⚙ gear icon inside the StarTrack rating panel on any movie page to open the Letterboxd settings for that user. Each user can:

  • Enter their Letterboxd username and enable hourly auto-sync — new ratings posted to Letterboxd import themselves into StarTrack within an hour
  • Upload their Letterboxd CSV export (Settings → Import & Export → Export Your Data on letterboxd.com) for a one-shot full-history import
  • Trigger a manual sync at any time via the "Sync now" button

StarTrack matches Letterboxd films to your library by title + year. Films not in your library are skipped and reported back on the user's sync status line. The hourly sync is a standard Jellyfin scheduled task — you can adjust its interval in Dashboard → Scheduled Tasks → StarTrack Letterboxd Sync.
